# 认知档案总览

> 本目录是 AI 理解「你是谁」的透镜。AI 在 `/digest` 和 `/analyze` 前都会读取这里的文件。

## 档案组成

- [[语言风格]]：表达 DNA（句式、词汇、节奏、幽默、确定性、引用）+ 常用句式/修辞、术语偏好、段落节奏。
- [[思维模型]]：心智模型（含三重验证）、决策启发式、推理模式、默认解决路径、智识谱系、内在张力。
- [[核心信念]]：根本立场、价值观排序、反模式、诚实边界。
